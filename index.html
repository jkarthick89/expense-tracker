Setup Guide

Complete these steps to get SplitMate running

🚀 One-time Setup Steps

1
Create a Google Cloud Project
Go to console.cloud.google.com → Create a new project (e.g. "SplitMate"). Enable the Google Sheets API and Google Drive API in APIs & Services → Library.
2
Set up OAuth 2.0 Credentials
APIs & Services → Credentials → Create Credentials → OAuth Client ID. Application type: Web application. Add your app's URL to Authorized JavaScript origins and Authorized redirect URIs. Copy the Client ID.
3
Add your Client ID to this file
Open this HTML file, find the variable CLIENT_ID near the top of the <script> and paste your OAuth Client ID.
4
Create your Google Sheet
Create a new Google Sheet in your Drive. The app will auto-create 3 tabs: Expenses, Members, and Trips. Copy the Sheet ID from the URL (the long string between /d/ and /edit).
5
Connect the Sheet
Go to Settings → paste your Sheet ID → click Save & Connect. The app will initialize all required tabs and columns automatically.
6
Host the app & share with friends
Host this single HTML file on GitHub Pages, Netlify, or Vercel (all free). Share the URL with friends — they'll sign in with their Google account. Access is controlled by the Members tab in your Sheet.

Expenses tab:
